Για να τρέξετε τα αρχεία χρειάζεστε Python 2 ffmpeg καθώς και τις Python βιβλιοθήκες numpy και PIL.

Δεν μπορούσα να βρω ικανοποιητικά uncompressed video, 
οπότε για το μέγεθος του κάθε βίντεο παίρνουμε το συνολικό μέγεθος των frames του 
(τα οποία βρίσκονται στο φάκελο temp, ο οποίος δημιουργείτε κατά την εκτέλεση).

Τα output αρχεία (βίντεο/εικόνα) βρίσκονται ήδη έτοιμα. 
Εάν τρέξετε τους αλγορίθμους, 
είναι πιθανό το ffmpeg να σας ζητήσει να τα ξαναγράψει. 
Επιλέξτε ναι ('y') εάν το θέλετε.

Οδηγίες για την εκτέλεση των προγραμμάτων βρίσκονται στους αντίστοιχους φακέλους.
Τα αρχεία "εργασια_6.16", "εργασια_6.17", "εργασια.8-18", 
στους αντίστοιχους φακέλους, περιέχουν απαντήσεις στις ερωτήσεις του βιβλίου.
